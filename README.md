# Blinkit Power BI Dashboard Template

This repository contains the **Blinkit.pbit** Power BI template designed for analyzing key performance metrics of Blinkit operations. The template provides an interactive report layout, pre-built visuals, and a structured data model to help users quickly connect their data and generate insights.

---

##  Features

- **Predefined report pages** for sales, product performance, delivery analytics, customer behavior, and more.
- **Ready-made visuals** such as KPI cards, line charts, bar charts, and slicers.
- **Data model structure** optimized for fast filtering and aggregation.
- **Custom theme** for consistent branding and clean visualization.
- **Parameter-driven setup** allowing users to connect their own datasets easily.

---

##  How to Use

1. Download the file:  
   **`Blinkit.pbit`**

2. Open the template in **Power BI Desktop**.

3. Enter or connect your data source when prompted.

4. Refresh the report to load your data into the pre-configured visuals.

5. Customize visuals if needed.

---

##  Requirements

- **Power BI Desktop** (latest version recommended)
- Compatible data tables (Sales, Orders, Products, Customers, etc.)
